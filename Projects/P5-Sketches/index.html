Javascript Sketches

Made with P5.js

This is a collection of the better sketches I made with javascript and p5.js. P5.js is a javascript library for rapid, creative coding. It is open source, and based on the Proccesing Language, a similar thing for Java. You can learn more about p5.js here.

This collection includes my oldest sketch, Wok 2d, which I made in Spring 2023. The others are newer, mainly from summer. You can see somewhat of a progression here. This is not a complete collection yet, I have several more to present. Please note any bugs, and enjoy! You can see what I'm working on now at my Open Proccesing Account.
